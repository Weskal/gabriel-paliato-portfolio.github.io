Projetos

Automação de Ingestão de CSV (Python + IMAP + S3)

Este projeto automatiza o processo de captura de arquivos CSV enviados por e-mail e os centraliza em um bucket da Amazon S3. O pipeline conecta-se a um servidor de e-mail IMAP, extrai e valida anexos CSV, processa-os usando bibliotecas Python como imaplib e email, e envia automaticamente os arquivos validados para o S3 com boto3. O objetivo é otimizar a ingestão de dados e eliminar o manuseio manual de arquivos.

BeanTracker (Node.js + React + MongoDB + Docker)

BeanTracker é uma aplicação full-stack desenvolvida para entusiastas de café registrarem, acompanharem e avaliarem grãos especiais. O sistema possui backend em Node.js, frontend SPA em React e banco de dados MongoDB, tudo containerizado com Docker e implantado numa instância EC2 na AWS. O projeto demonstra práticas modernas de desenvolvimento web, design de API, armazenamento persistente e fluxos de implantação com contêineres.

Vexus Data Pipeline (Airflow + Python + SQL Server)

Este projeto implementa um pipeline de dados totalmente automatizado para uma loja de roupas fictícia chamada Vexus. Scripts em Python geram dados sintéticos de pedidos de vendas e os enviam para o SQL Server, enquanto o Apache Airflow orquestra as etapas de extração, transformação e carga. O pipeline demonstra automação ponta a ponta, agendamento, monitoramento e armazenamento de dados para cargas analíticas.

CODVUNS — Sistema de Análise de Vulnerabilidades de Código (Projeto de TCC)

O CODVUNS é uma ferramenta de análise estática criada para detectar vulnerabilidades de segurança em código-fonte. Ela combina correspondência de padrões (regex), inspeção de árvore sintática abstrata (AST) e, futuramente, aprendizado de máquina para identificar estruturas de código inseguras com alta precisão. Desenvolvido como projeto de trabalho de conclusão de curso, o sistema automatiza a varredura de segurança e fornece avaliações consistentes e reproduzíveis da qualidade e do risco do código.

Construção de uma arquitetura de microsserviços multilíngue para uma loja de roupas fictícia. Agente Analista de Dados com LLM — Python + SQL + LangChain

Este projeto implementa um assistente de análise de dados que permite a usuários não técnicos consultarem bancos de dados SQL Server usando perguntas em linguagem natural. O sistema utiliza LangChain e OpenAI para: Compreender perguntas em linguagem natural (inglês/português); Gerar consultas SQL otimizadas com base no esquema do banco de dados; Executar as consultas no banco com validações de segurança; Analisar os resultados e criar uma resposta informativa sobre os dados.
