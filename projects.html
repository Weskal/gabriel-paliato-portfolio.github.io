Projects

Automation CSV Ingestion (Python + IMAP + S3)

This project automates the process of capturing CSV files sent via email and centralizing them in an Amazon S3 bucket. The pipeline connects to an IMAP email server, extracts and validates CSV attachments, processes them using Python libraries such as imaplib and email, and automatically uploads the validated files to S3 using boto3. The goal is to streamline data ingestion and eliminate manual file handling.

BeanTracker (Node.js + React + MongoDB + Docker)

BeanTracker is a full-stack application designed for coffee enthusiasts to record, track, and evaluate specialty beans. The system features a Node.js backend, a React SPA frontend, and a MongoDB database, all containerized with Docker and deployed to EC2 instance in AWS environment. The project showcases modern web development practices, API design, persistent storage, and containerized deployment workflows.

Vexus Data Pipeline (Airflow + Python + SQL Server)

This project implements a fully automated data pipeline for a fictional clothing store called Vexus. Python scripts generate synthetic sales order data and ingest it into SQL Server, while Apache Airflow orchestrates extraction, transformation, and loading workflows. The pipeline demonstrates end-to-end automation, scheduling, monitoring, and data storage for analytical workloads.

CODVUNS — Code Vulnerability Analysis System (TCC/End of Course Paper Project)

CODVUNS is a static analysis tool built to detect security vulnerabilities in source code. It combines pattern-matching (regex), abstract syntax tree (AST) inspection, and soon will also be implemented machine learning to identify insecure code structures with high accuracy. Developed as my final thesis project, the system automates security scanning and provides consistent, reproducible assessments of code quality and risk.
